# M0M3NTUM — Windows Build

**Download:** [M0M3NTUM-Windows.zip](M0M3NTUM-Windows.zip) (38MB)

## How to Run

1. Download `M0M3NTUM-Windows.zip`
2. Extract the zip file
3. Run `M0M3NTUM-Windows.exe`
4. No installation needed!

## Controls

- **Arrow Keys / WASD** — Move
- **Space** — Jump (hold for higher jump)
- **W** — Wall Jump (when touching wall)
- **Shift** — Dash (1 per jump, resets on landing/wall)
- **Mouse Click** — Grapple (upward-only, Spider-Man style)
- **S / Down** — Crouch
- **Esc** — Pause / Level Select
- **R** — Restart Level

## System Requirements

- **OS:** Windows 10 or later (64-bit)
- **RAM:** 2GB minimum
- **GPU:** Any DirectX 11 compatible
- **Disk:** 150MB free space

## Features

- 10 challenging levels
- N++/Celeste-inspired physics
- Combo system (wall jump → dash → grapple)
- Procedurally generated audio
- Instant respawn, no death animations
- Best times tracked per level
